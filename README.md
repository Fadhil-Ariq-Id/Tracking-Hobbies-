## Features & Technologies Used

Proyek ini mencakup fitur-fitur utama dan penerapan teknologi berikut:

*   **Architecture:** Implemented **MVVM (Model-View-ViewModel)** pattern for robust state management and separation of concerns.
*   **Clean Architecture:** Structured the project using principles of Clean Architecture (Domain, Data, Presentation layers) for scalability and testability.
*   **Supabase Client:** Integrated the Supabase client for backend services.
*   **Authentication:** Managed user authentication flows using Supabase Auth.
